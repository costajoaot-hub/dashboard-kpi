# Sustainability Project Dashboard Generator

A professional data visualization tool developed to track and present Key Performance Indicators (KPIs) for educational sustainability projects. This script transforms multi-dimensional metrics into a high-resolution, boardroom-ready dashboard.

## Key Visualizations
* **Capacity Building:** Tracks trainer-to-faculty ratios over time using grouped bar charts.
* **Curriculum Integration:** Dual-axis chart combining absolute course counts with percentage-based awareness metrics.
* **Resource Auditing:** Overlays digital content creation trends on top of formal training modules.
* **Community Outreach:** Multi-line tracking of workshops, campaigns, and student-led initiatives.

## Requirements
* **Python 3.x**
* **Matplotlib:** For high-quality chart rendering.
* **NumPy:** For efficient numerical data handling.
